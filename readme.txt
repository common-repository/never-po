Translate text in backend WP mode without using po & mo files, and poedit mechanism.

== Description ==

= Возможности =
Данный плагин позволит вам переводить любой тескт в консоли администартора, который не имеет оберток в функцию перевода. Плагин предоставляет Вам возможность составить словарь
и указать в каких разделах сайта выполнять перевод введенных в него фраз.
Поддерживается режим WP мультисайт. Каждое вхождение словарной фразы может быть переведено: только в конкретном экране бэкэнда или везде, только в конкретном блоге WP мультисайт или во всех блогах.
Все переводы учитывают текущую локаль сайта и работают только в ней, т.е. вы можете выполнять перевод РАЗДЕЛЬНО для всех существующих на сегодня локалей WP

== Installation ==

1. Распакуйте архив с плагином
2. Загрузите папку с плагином в директорию `/wp-content/plugins/` на вашем сервере
3. Активируйте плагин Never PO в разделе Плагины в панели администрирования WordPress
4. Активируйте режим ПЕревод и начните наполнять словарь
